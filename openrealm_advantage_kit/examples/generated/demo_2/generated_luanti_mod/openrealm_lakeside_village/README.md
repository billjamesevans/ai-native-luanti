# Cozy Lakeside Village

Creates a bounded starter cozy lakeside village with custom nodes, preview command, build command, and rollback metadata.

Generated by OpenRealm Creator Kernel. Install in a disposable test world first.

## Commands

- `/or_preview`
- `/or_build <structure_name>`
- `/or_rollback_last`

## Safety

- AI did not directly mutate the world.
- Changes are template-generated from a data-only plan.
- Structures are queued through the AI runtime import task queue when available.
- Rollback metadata is stored by the AI runtime before structure placement.
